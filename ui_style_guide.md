# UI Aesthetic: Urban Topographical (The "Signal" Style v2)

This style guide refines the SIGNL visual direction to a high-contrast, premium urban aesthetic.

## Visual Tokens
- **Background**: Blend of ethereal topographical cloud lines overlaying a high-resolution, muted urban skyline. The atmosphere should feel "high-altitude" and "sovereign."
- **Imagery**: Ultra-clean, sharp architectural photography. Avoid clutter. Use large, expansive vistas.
- **Typography**: 
  - **Headings**: Elegant Serif (e.g., *Instrument Serif*) for brand authority.
  - **Body/UI**: Crisp Sans-Serif (*Geist* or *Inter*) for operational efficiency.
- **Depth**: Soft glassmorphism. Components should feel like they are floating above the city.
- **Buttons**: Minimalist but high-fidelity. No bulky borders. Use subtle micro-gradients and crisp hover states.

## Key Components
- **The "Signal" Hero**: Serif heading floating over a topographical cityscape.
- **Urban Cards**: Dark, translucent surfaces (rgba(13,17,23, 0.85)) with sharp 1px internal highlight borders.
- **Action Buttons**: Sleek, sharp-edged buttons with high-contrast text and SIGNL Amber accents on hover.

## Key Components
- **The "Signal" Hero**: Large serif heading centered on a spacious, cloudy background.
- **Workflow Cards**: Numbered (01, 02, 03) cards with subtle gradients and crisp typography.
- **Inputs**: Minimal, borderless or soft-bordered inputs that float on the background.
